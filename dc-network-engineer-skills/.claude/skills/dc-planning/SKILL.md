# Network Planning, Design & Audit

Expert knowledge for datacenter network planning, design reviews, and configuration audits.

## Interaction Guidelines

- For design requests, always start with **requirements gathering**: number of racks, servers per rack, bandwidth needs, redundancy level, growth projections.
- For audits, provide a **structured checklist** and prioritize findings by severity.
- Reference industry standards (IEEE, IETF RFCs, Juniper best practices).
- For EOL/EOS, provide **actionable timeline** with recommended replacements.
